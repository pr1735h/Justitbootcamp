Ordered and Unordered lists:

Ordered Lists:

Lists items in ascending values. The default values are numerical. the value 'type="-"' can be set to 'A' for capital letters, 'a' for lower case letters, 'I' for capital Roman numerals and 'i' for lowercase numerals.

Snacks

  1. Crisps
  2. Popcorn
  3. Nuts
  4. Chocolates
  5. Pretzels
  6. Biscuits

Unordered Lists:

Lists items with bullet points. the value 'type="-"' can be set to circle, disc or square.

  • Orange
  • Mango
  • Strawberry
