Flex Basis

Flex lets all flexible items be the same length, regardless of its content.
Flex basis is a property that sets the initial length of flex items.
Flex direction explains itself.
Flex flow can be used to reverse items.
Flex grow or shrink allow items to scale up or down in relation to another item.
Flex wrap allows items to wrap if they need to be wrapped.
