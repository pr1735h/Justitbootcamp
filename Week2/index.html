Cascading Style Sheets make websites look nicer than using only html.
Cascading means it starts from the top and goes down the page to apply styles, giving priority to the latest style applied to an element.
CSS can be used as 'inline' or 'internal' within a html file but refering to an 'external' style sheet is better for a multi page site or page with a lot of styling.
